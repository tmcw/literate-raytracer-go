# go raytracer

![](http://farm8.staticflickr.com/7314/11142296544_80422ecfbb_z.jpg)

A working raytracer in Go, a port of [literate-raytracer](http://macwright.org/literate-raytracer/)
from JavaScript.

## usage

    go run main.go
